# c_cgi

C言語でCGIプログラムを書いてみる
(C言語の勉強も兼ねつつ、docker触るのが久しぶりなのでdockerの復習もする)

# How To Use

git clone
docker-compse build
docker-compose up -d

(いまのdockerだと docker compose build とdocker compose up -d )

その他便利なコマンド
docker-compose exec web bash
(コンテナのなかに入る)
